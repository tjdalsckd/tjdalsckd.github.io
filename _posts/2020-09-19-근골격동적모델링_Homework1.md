### Kane's Method의 특성
1. 벡터기반(vector-based) 방법으로 3D 분석에 용이
2. Vector cross product와 dot product를 적극적으로 활용
3. Generalized force를 이용하여 운동 방정식 기술
4. Generalized inertia force ${F}^{*}_{r}$

$\vec { F } _ { B } + \vec { F } ^ { B ^ { * } } = 0$
